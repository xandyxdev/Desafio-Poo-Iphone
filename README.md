# Desafio-Poo-Iphone 
# Modelagem e Diagramação de um Componente iPhone 

Neste desafio, fui responsável por modelar e diagramar a representação UML do componente iPhone, abrangendo suas funcionalidades como Reprodutor Musical, Aparelho Telefônico e Navegador na Internet. Além da modelagem, também implementei o código em Java com base no diagrama criado.
